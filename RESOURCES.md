## 📊 监控与分析

### 使用统计

```bash
# 查看今日使用量
openclaw stats today

# 查看本月使用量
openclaw stats month

# 查看模型分布
openclaw stats models
```

### 日志查看

```bash
# 实时日志
openclaw logs -f

# 错误日志
openclaw logs --level error

# 最近100行
openclaw logs -n 100
```
