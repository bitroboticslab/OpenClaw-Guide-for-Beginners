### 2. 智能依赖检测

每个脚本自动检测依赖：
```bash
# 检测 Node.js
if command -v node &> /dev/null; then
    NODE_VER=$(node -v | cut -d'v' -f2 | cut -d'.' -f1)
    if [ "$NODE_VER" -ge 22 ]; then
        echo "[√] Node.js 已安装: $(node -v)"
        return 0
    fi
fi
```
