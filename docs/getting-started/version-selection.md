# 版本选择说明
## 🎯 版本选择原则
- **新手用户**：选择最新稳定版，跟随官方更新即可
- **生产环境用户**：选择已经发布至少1周的稳定版本，避免踩最新版本的bug
- **尝鲜用户**：可以使用预览版或开发版，体验新功能
## 📋 版本支持周期
| 版本类型 | 支持周期 | 说明 |
|---------|---------|------|
| 稳定版 | 12个月 | 官方会修复bug和安全漏洞，推荐大多数用户使用 |
| LTS长期支持版 | 24个月 | 针对生产环境用户，提供更长时间的支持，不会频繁引入新功能 |
| 预览版 | 2周 | 测试新功能使用，不建议在生产环境使用 |
## 🆚 各版本对比
### 最新稳定版：2026.4.14 ✅ 推荐
#### 优势
- 包含最新的安全强化功能，沙箱模式默认开启，大幅降低误操作风险
- 支持飞书官方一键对接，无需手动配置API，5分钟即可完成对接
- 性能大幅优化，内存占用降低40%，查询速度提升200%
- 支持所有最新功能：多Agent协同、多媒体生成、记忆系统等
#### 适配系统
- WSL：✅ 完美支持
- Windows：✅ 支持，建议开启沙箱模式
- macOS：✅ 完美支持
- Linux：✅ 完美支持
- Docker：✅ 官方镜像已更新
### 历史版本：2026.4.1
#### 优势
- 相对成熟，经过1个月的用户验证，bug较少
#### 劣势
- 缺少最新的安全强化功能，误操作风险更高
- 不支持飞书一键对接，需要手动配置API
### 历史版本：2026.3.1
#### 优势
- 非常成熟，经过3个月的大规模验证，稳定性极高
#### 劣势
- 缺少很多新功能，后续不再提供新功能更新
- Node.js版本要求较低，性能较差
## 🚀 升级指南
### 从2026.3.x/2026.4.x升级到最新版本
完全向下兼容，无需修改配置，直接执行升级命令即可：
```bash
# 全局升级
npm install -g openclaw@latest
# 验证版本
openclaw --version
# 重启网关
openclaw gateway restart
```
### 从2026.2.x及更早版本升级
建议先备份配置文件，然后按照最新教程重新安装，避免兼容性问题。
