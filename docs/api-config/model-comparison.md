## 🎯 按场景推荐

### 编程开发

**推荐**: 火山引擎方舟 或 智谱GLM Coding

**原因**:
- 专为编程场景优化
- 支持主流开发工具
- 上下文长度充足
- 代码生成能力强

**选择建议**:
- 首选 **火山引擎**：多模型支持，性价比高
- 次选 **智谱GLM Coding**：编程优化工具丰富

---

### 日常对话

**推荐**: 硅基流动（免费）或 火山引擎

**原因**:
- 响应速度快
- 对话自然流畅
- 成本低或免费

**选择建议**:
- 新手：先用硅基流动免费额度
- 进阶：升级到火山引擎Coding Plan

---

### 长文本处理

**推荐**: 智谱GLM Pro

**原因**:
- 支持长上下文（128K+）
- 文档总结能力强
- 适合读书、论文分析

---

### [Agent开发](../advanced/skills.md)

**原因**:

## ⚙️ 在OpenClaw中配置

### 单模型配置

```json
{
  "models": {
    "providers": [
      {
        "providerId": "bailian",
        "apiKey": "你的API密钥",
        "models": [
          {
            "id": "qwen-plus",
            "primary": true
          }
        ]
      }
    ]
  }
}
```

### 多模型配置（推荐）

```json
{
  "models": {
    "providers": [
      {
        "providerId": "bailian",
        "apiKey": "你的API密钥",
        "models": [
          {
            "id": "qwen-plus",
            "primary": true,
            "tags": ["general", "chat"]
          },
          {
            "id": "qwen-coder-plus",
            "tags": ["code", "programming"]
          }
        ]
      }
    ]
  },
  "agents": {
    "defaults": {
      "model": {
        "primary": "bailian/qwen-plus"
      },
      "coding": {
        "model": "bailian/qwen-coder-plus"
      }
    }
  }
}
```
