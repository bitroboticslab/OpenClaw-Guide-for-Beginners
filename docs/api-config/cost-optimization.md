## ⚙️ 优化配置

### 1. 智能模型切换

根据任务类型自动选择模型：

```json
{
  "agents": {
    "defaults": {
      "model": {
        "primary": "bailian/qwen-turbo",
        "coding": "bailian/qwen-coder-plus",
        "reasoning": "bailian/qwen3-max"
      }
    }
  }
}
```

**优势**:
- 简单对话用低成本模型
- 编程任务用专业模型
- 推理任务用强模型

---

### 2. 限制上下文长度

减少不必要的Token消耗：

```json
{
  "models": {
    "maxTokens": 2000,
    "temperature": 0.7
  }
}
```

---

### 3. 缓存常用回复

减少重复请求：

```json
{
  "agents": {
    "caching": {
      "enabled": true,
      "maxAge": 3600
    }
  }
}
```

---

### 4. 设置使用限额

避免意外超额：

```json
{
  "models": {
    "rateLimiting": {
      "maxRequestsPerDay": 1000,
      "maxTokensPerDay": 100000
    }
  }
}
```
