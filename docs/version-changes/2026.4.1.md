# OpenClaw 2026.4.1 版本变更说明
发布时间：2026-04-01
## 功能更新
- ✅ Node.js版本最低要求从22升级到24，性能提升40%，安全性大幅增强
- ✅ 新增macOS桌面应用一键安装包，支持图形化配置
- ✅ 新增Android移动节点支持，手机也可以运行OpenClaw节点
- ✅ 多平台对接新增QQ、LINE、Matrix、Teams支持
- ✅ 飞书插件官方支持升级，内置覆盖IM、日历、任务、多维表、云文档、云盘、wiki、表格等全场景能力，无需额外配置
### 命令变更
- 废弃`openclaw service`系列命令，统一使用`openclaw gateway`命令
- 配置向导命令从`openclaw onboard`优化为`openclaw init`（兼容旧命令）
- 新增`openclaw doctor --fix`命令，支持自动修复常见配置问题
### 兼容性说明
- 完全向下兼容2026.3.x版本配置，无需修改即可升级
- Node.js 22版本还会继续支持6个月，建议尽快升级到24版本获得更好性能
