# OpenClaw 2026.4.14 版本变更说明
发布时间：2026-04-14
## 重大变更
### 安全强化
- ✅ 新增沙箱运行模式，默认限制文件系统访问范围，避免误操作删除用户重要文件
- ✅ 权限最小化改造，所有敏感操作需要用户显式授权
- ✅ Windows平台运行新增安全提醒，推荐使用WSL模式避免系统级风险
### 功能更新
- ✅ 飞书官方一键对接功能正式发布，扫码即可完成机器人创建与配置，无需手动申请API权限
- ✅ ACP子代理框架升级，支持多代理并行执行任务，性能提升300%
- ✅ 记忆系统新增向量索引自动优化功能，查询速度提升200%，内存占用降低40%
- ✅ 新增图片/视频生成功能内置支持，无需额外配置插件
- ✅ systemd服务管理优化，新增`openclaw gateway enable/disable`命令，支持开机自启配置
### 命令变更
- 旧命令：`openclaw onboard` → 新命令：`openclaw init`（兼容旧命令）
- 旧命令：`openclaw gateway start` → 新增别名支持：`openclaw start`
- 新增命令：`openclaw gateway enable` 配置开机自启
- 新增命令：`openclaw gateway disable` 取消开机自启
### 兼容性说明
- 完全向下兼容2026.3.x/2026.4.x版本配置，无需修改即可升级
- 沙箱功能默认开启，如果需要访问系统级目录可以在配置中关闭（不推荐）
